SHORT DESCRIPTION
    AsgGroupLegacy assists in performing certain generic tasks within PowerShell 5.1 using functions.


LONG DESCRIPTION
    AsgGroupLegacy provides assistance in performing generic tasks within PowerShell 5.1, with the help of advanced functions.


DETAILED DESCRIPTION
    This module helps in performing certain generic tasks that are sometimes required within PowerShell 5.1, such as the creation of encrypted password files and utilising them in creating credential objects.
